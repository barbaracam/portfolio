Covid Date

Covid-Date is a website offering some wonderful suggestions about different activities to do with your partner, roommate, or members of the same household. Covid-Date will provide different recipes and suggested movie lists to organize a lovely evening with the members of your household. The website uses two API: Spoonacular & Trakt API

Password Manager

Password Manager is an offline application that allows user/s to store and manage their password for other local applications or web services. The application will support client/s to generate and store complex and simple passwords in a database.

Technologies:PHP, JavaScript, HTML, CSS, MySQL
